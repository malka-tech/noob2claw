# 🚀 Noob2Claw – Agentic Webinterface Framework

> **Ein vollständiges Architektur-Framework für die Entwicklung moderner Webanwendungen mit KI-Agenten wie OpenClaw.**

---

## Was ist das?

Dieses Repository enthält **keinen fertigen Quellcode**, sondern die komplette Projektdokumentation, mit der ein KI-Agent ein vollständiges Webinterface entwickeln kann.

Anstatt der KI nur einen einzelnen Prompt zu geben, erhält sie eine vollständige Softwarearchitektur mit:

- Architekturvorgaben
- Verzeichnisstruktur
- Datenbankschema
- Coding Rules
- API- und MCP-Konzept
- Benutzer- und Rechteverwaltung
- UI- und Designsystem
- CSS-Standards
- Formular- und Tabellenstandards
- zentralem Startprompt

Dadurch entsteht ein konsistentes und langfristig wartbares Projekt.

# 🤖 Wie funktioniert das?

```text
Markdown-Dokumentation
            │
            ▼
      OpenClaw Agent
            │
            ▼
Entwicklung der kompletten Anwendung
            │
            ▼
PHP + MariaDB + Apache
            │
            ▼
Lauffähiges Webinterface
```

Der KI-Agent liest zuerst sämtliche Markdown-Dateien ein und entwickelt anschließend das komplette Projekt anhand dieser Vorgaben.

---

# 💡 Ziel

Das Ziel ist es, reproduzierbare und qualitativ hochwertige Software mit KI zu entwickeln.

Die Dokumentation definiert unter anderem:

- Projektarchitektur
- Sicherheitsregeln
- Datenbankdesign
- Business-Logik
- UI-Standards
- CSS-Regeln
- Formulare
- Tabellen
- Navigation
- API
- MCP

Dadurch entwickelt die KI nicht einfach nur Code, sondern hält sich an verbindliche Entwicklungsrichtlinien.

# ⚠️ Hinweis

Dieses Repository ist **kein fertiges Framework**, sondern eine vollständige Entwicklungsgrundlage für KI-Agenten.

Die eigentliche Stärke liegt nicht im Quellcode, sondern in der Kombination aus:

- Architektur
- Standards
- Regeln
- Prompts
- Designsystem
- Entwicklungsprozess

Damit kann ein KI-Agent ein komplettes, professionelles Webinterface nahezu selbstständig entwickeln.
